# ⚠️ ДИАГНОСТИЧЕСКАЯ ПРОШИВКА - BLK-fix-thumb-cluster

**Это тестовая прошивка для диагностики тамб кластера МК2!**

## Цель диагностики

Определить реальную распайку тамб кластера МК2 (черной клавиатуры).

## Что сделано в этой прошивке

1. **Matrix transform**: Все 8 колонок доступны для тамб кластера (row 4)
2. **Keymap**: На каждую позицию тамб кластера назначена цифра:
   - **Левая половина** (слева направо): 0, 1, 2, 3
   - **Правая половина** (слева направо): 5, 6, 7, 8

## Инструкция по тестированию

### Шаг 1: Прошей клавиатуру

1. Скачай артефакт `firmware_BLK_SVG` из GitHub Actions
2. Прошей **обе половины** МК2
3. Перезагрузи клавиатуру

### Шаг 2: Тестирование левой половины

Открой блокнот и нажми **каждую клавишу** в тамб кластере левой половины:

**Левая тамб (всего 4 клавиши)**:
- Клавиша 1 (самая левая): _____
- Клавиша 2: _____
- Клавиша 3: _____
- Клавиша 4 (самая правая): _____

### Шаг 3: Тестирование правой половины

Нажми **каждую клавишу** в тамб кластере правой половины:

**Правая тамб (всего 4 клавиши)**:
- Клавиша 1 (самая левая): _____
- Клавиша 2: _____
- Клавиша 3: _____
- Клавиша 4 (самая правая): _____

### Шаг 4: Сообщи результаты

Напиши мне **какая цифра сработала при нажатии каждой клавиши**.

Например:
```
Левая: 
- Клавиша 1: не работает
- Клавиша 2: цифра 1
- Клавиша 3: цифра 2
- Клавиша 4: цифра 3

Правая:
- Клавиша 1: цифра 5
- Клавиша 2: цифра 6
- Клавиша 3: не работает
- Клавиша 4: цифра 8
```

## Технические детали

### Matrix Transform

```
Row 4 (thumb cluster):
  Левая: RC(4,0) RC(4,1) RC(4,2) RC(4,3)
  Правая: RC(4,13) RC(4,14) RC(4,15) RC(4,8)
```

### Col-gpios маппинг (8 колонок)

- Col 0: P0.2
- Col 1: P0.29
- Col 2: P0.9
- Col 3: P1.0
- Col 4: P0.11
- Col 5: P0.10
- Col 6: P1.13
- Col 7: P1.4

### При нажатии клавиши срабатывает цифра = номер колонки

Например:
- Если при нажатии клавиши срабатывает цифра "1" → это Col 1 (P0.29)
- Если срабатывает цифра "3" → это Col 3 (P1.0)
- Если срабатывает цифра "5" → это Col 5 (P0.10)
- Если срабатывает цифра "8" → это Col 8 (правая половина col 0)
